# Kubernetes (K8s) Learning Path

A comprehensive guide to Kubernetes, from fundamentals to production-ready deployments.

## 📚 Contents

1. **[Fundamentals](01_fundamentals.md)** - Core concepts, architecture, and components

3. **[Services and Networking](03_services_and_networking.md)** - Services, Ingress, Network Policies
4. **[Configuration and Secrets](04_configuration_and_secrets.md)** - ConfigMaps, Secrets, environment variables
5. **[Storage](05_storage.md)** - Volumes, PersistentVolumes, StorageClasses

9. **[Interview Questions](09_interview_questions.md)** - Common interview questions and answers

## 🎯 Learning Path

### Beginner
- Start with Fundamentals
- Learn about Pods and basic workloads
- Understand Services and basic networking

### Intermediate
- Deep dive into Configuration management
- Learn Storage concepts
- Understand Security basics

### Advanced
- Master Observability
- Study Production patterns
- Practice with real-world scenarios

## 🛠️ Prerequisites

- Basic understanding of containers (Docker)
- Linux command line knowledge
- Basic networking concepts
- YAML syntax familiarity

## 🚀 Getting Started

1. Install kubectl: `brew install kubectl` (macOS)
2. Set up a local cluster: Minikube, Kind, or Docker Desktop
3. Follow the learning path sequentially
4. Practice with hands-on examples
